# OTP Generator

A simple OTP generator built with HTML, CSS, and JavaScript.

## Features
- Generates a 6-digit OTP.
- Validates the mobile number before generating OTP.

## How to Use
1. Enter a 10-digit mobile number.
2. Click "Generate OTP" to get your OTP.
